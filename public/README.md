# Public Assets

This directory contains static assets for the Electron app.

## Required Files

- **icon.ico** - Windows application icon (256x256 recommended)
- **icon.png** - Tray icon and general use (32x32 or 64x64)
- **icon.icns** - macOS application icon (for future)

## Creating Icons

1. Design a 1024x1024 PNG logo
2. Use a tool like [RealFaviconGenerator](https://realfavicongenerator.net/) to create all formats
3. Place the files in this directory

## Temporary Workaround

Until you add icons, the app will use a default placeholder.
