# Add screenshots here

This directory is for documentation assets like screenshots and diagrams.

## Recommended Screenshots

1. **screenshot-idle.png** - Main overlay interface in idle state
2. **screenshot-executing.png** - Agent actively executing a task
3. **screenshot-steps.png** - Step visualization showing Analyze → Plan → Execute

## How to capture

1. Run HyperOS: `./start.bat`
2. Press `Win + Shift + S` to open Snipping Tool
3. Capture the overlay window
4. Save images to this folder
